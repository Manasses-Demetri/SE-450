Sprint 2:
Deadlock Draft & Team Composition Analyzer

This project is a Java command line application that assists players during the drafting phase of a Deadlock match. Users will create two teams by selecting heroes for the Blue and Red sides. As heroes are selected, the application will maintain each team's roster and eventually analyze the overall team composition.

The final version of the project will evaluate characteristics such as damage, survivability, crowd control, mobility, and team synergy. It will also recommend hero picks based on the current draft, identify strengths and weaknesses of each team, and provide a detailed draft report explaining why a composition is strong or weak. Throughout the semester, the project will incorporate multiple object oriented design patterns to keep the code modular, maintainable, and easy to extend with new heroes and analysis features. I would also love to add item reccomendations for the users specific pick if I have time.
